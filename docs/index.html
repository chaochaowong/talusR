A lightweight R toolkit for Talus Bio proteomics data analysis
View Vignette

Key Features

The talusR package provides a workflow for Talus mass-spectrum data—import, QC, differential analysis, and visualization
(requires DIA‐NN inputs and matching metadata).

  • Import & Structure: Import data into TalusDataSet or TalusDataSetList S4 objects.
  • QC: PCA and multivariate checks for clustering & outliers.
  • Differential Analysis: Wraps limma & matrixTests for hypothesis testings.
  • Visualization: PCA, volcano, and per‐protein plots via ggplot2.
  • Bioconductor Ready: Built on SummarizedExperiment and S4Vectors.
